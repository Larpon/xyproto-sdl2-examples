SDL2, C++17 and CXX
===================

Requirements
------------

* [CXX](https://github.com/xyproto/cxx)
* Compiler that supports C++17 (recent version of `g++` or `clang++`)
* SDL2

Building with C++17, SDL2 and cxx
---------------------------------

    cxx

Building and running
--------------------

    cxx run

Cleaning up the binary file and build directory
-----------------------------------------------

    cxx clean
